In current WEP porting stage we are able to run XFree minimalistic server named KDrive. It is operating on framebuffer (already supported) and as input device it uses touchpad supported via "tslib".

Building instructions

Uncompress all files from XFree project. Create file name "host.def" in './config/cf/' directory containing 
#define KDriveXServer		YES
#define TinyXServer		YES
#define XfbdevServer		YES
#define TouchScreen		YES
#define HasTsLib		YES
If your interest is only to build server, not all supporting stuff add this line into host.def 
#define BuildServersOnly	   YES
Typing 
make World CROSSCOMPILEDIR=/cross-compiler-bin

Just note "/cross-compiler-bin" is directory containing cross-compiler binaries or links to cross compiler without prefix like "arm-linux-".

Instalation could be complete by 
make install CROSSCOMPILEDIR=/cross-compiler-bin DESTDIR=../image

Result server name is "Xfbdev" and to get fully operating server you also have to cross-compile so called 'tslib' located in CVS of arm.linux.org.uk
